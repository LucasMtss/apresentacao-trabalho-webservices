Trabalho de Webservice

API de notícias com integração com Twitter e análise de sentimento

O Projeto

Este projeto tem como objetivo fixar os conhecimentos sobre criação e consumo de APIs de a comunicação entre elas utilizando uma linguagem de interoperabilidade, no caso do projeto, o JSON.

A ideia do software é criar uma API que irá consumir uma outra API de notícias, e a partir das notícias recebidas pesquisar por tweets relacionados ao assunto e em seguida realizar uma análise de sentimento dos mesmos para entender os usuários estão comentando bem ou mal sobre o assunto.

O resultado desse processamento é consumido por uma página web e é apresentado para o usuário. As notícias são segmentadas por categorias.

Tecnologias Utilizadas

  • Para a construção da API foi utilizada a linguagem Python e a biblioteca Flask.
  • Para consumo da API foi utilizado React, uma biblioteca de javascript destinada ao frontend.
